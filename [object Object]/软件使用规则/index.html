软件使用规则

发表于 2018-07-30   |   更新于 2018-09-12 | 分类于 资源集 | 本文总阅读量 次
字数统计: | 阅读时长 ≈

我非常喜欢用起来很爽的工具，那会让我再工作和学习的时候心情愉悦。我将把我工作、生活中使用起来非常爽的软件、网站与APP分享出来，其中会包括一些软件的破解方法。

首先声明，我并不鼓励使用破解软件，但是我不想上万元人民币的门槛成为阻碍我前进的障碍，所以破解的资源希望大家不要肆意传播，自己使用就好。

其次，很多公司会有学生计划，如果看到我的博客的你还不知道该如何去找，希望这个能对你有所帮助：如何赚回学费！高校edu邮箱完整使用方式！。

最后，跟个风，我推荐大家使用开源的解决方案，在我推荐的软件中，大家可以将在搜索引擎中搜索“软件名+ 开源解决方案”，就能搜索到相应的结果。

工欲善其事必先利其器，我将主要使用Resilio Sync 和 MEGA Sync 作为传输工具。

因为墙内不能直接访问MEGA，如果不想选择科学上网的朋友，可以下载MEGA的桌面版或插件。MEGA Chrome插件 MEGA Firefox插件 MEGA for Windows

坚持原创技术分享，您的支持将鼓励我继续创作！
